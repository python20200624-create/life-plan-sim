# ⚡ LifePlan Simulator Pro — Mobile V6

あなたの人生を「見える化」する、本格ライフプラン・シミュレーター

---

## 📌 商品概要

LifePlan Simulator Pro は、スマートフォンのブラウザだけで動く、完全オフライン対応のライフプラン・シミュレーターです。  
年収・支出・住宅・教育費・資産運用・年金……あらゆる要素を入力するだけで、老後まで見通したキャッシュフローを即座にグラフ化。  
1,000回のモンテカルロ・シミュレーションによる資産生存確率の算出や、ルールベースの自動診断スコアリングまで搭載した、本格仕様のファイナンシャルツールです。

> 🔒 完全ローカル動作 — すべての計算はブラウザ上で完結。あなたの資産データがサーバーに送信されることは一切ありません。

---

## ✨ 主な機能

### 👨‍👩‍👧‍👦 家族設定
- 独身 / 既婚の選択
- 配偶者の年収・退職年齢・退職金の設定
- 最大3人までの子供の教育プラン設定（公立 / 私大文系 / 私大理系 / 私大医歯薬）

### 💰 支出設定
- 基本生活費（年額）
- 住宅戦略：賃貸 / 持家（ローン返済中） / 将来購入
  - 住宅ローン：元利均等返済の正確な計算（銀行実務準拠）
  - 修繕積立金・固定資産税も考慮
- 車両：最大2台、買い替えサイクル・終了年齢を自由設定
- ライフイベント：旅行・冠婚葬祭・リフォームなど自由に追加（無制限）

### 📈 資産運用
- 現預金・投資元本・月額積立額の設定
- 固定 / 変動利回りモード（ボラティリティ設定可能）
- インフレ率の考慮
- 生活防衛資金プロテクション（現預金の最低維持ラインを自動管理）
- キャッシュクッション戦略（退職前の段階的資産売却）

### 📊 診断結果
- 資産推移グラフ（Chart.js による高品質チャート）
  - ピンチ操作でズーム・スワイプでパン（モバイル最適化済み）
  - 全画面モード：HUD付きの詳細分析表示
- 自動診断スコア（最終資産額に基づくルールベース判定・リングインジケーター表示）
- 年齢別 資産生存確率（変動モード時、1,000回モンテカルロ試行）
- 最終資産額・資産枯渇年齢の即時表示

### 💾 データ管理
- ローカル保存（localStorage）
- JSONエクスポート / インポート対応（端末間の移行に便利）

## 🧮 計算エンジンの特長

| 項目 | 内容 |
|:---|:---|
| 教育費 | 文科省データ準拠 × リスク係数1.2 (幼稚園〜大学院) |
| 公的年金 | 老齢基礎年金 + 老齢厚生年金の二階建て構造を再現 |
| 住宅ローン | 元利均等返済（銀行実務計算式） |
| 車両費 | 買い替えサイクルに基づく自動計上 |
| モンテカルロ法 | 1,000回試行による確率的資産シミュレーション |
| 生活防衛資金 | 赤字時の資産取り崩し優先順位を忠実に再現 |

## 🚀 使い方

### かんたん3ステップ

1. ファイルを展開 — ダウンロードしたZIPを解凍します
2. `index.html` を開く — スマートフォンまたはPCのブラウザで開きます
3. 情報を入力して実行 — ヘッダーの更新ボタン（🔄）をタップでシミュレーション開始

> 💡 サーバー不要。HTMLファイルをブラウザで開くだけで、すべての機能がすぐに動きます。

### 推奨環境
- スマートフォン: iOS Safari / Android Chrome（推奨）
- PC: Chrome / Edge / Firefox（最新版）
- インターネット接続は不要です

## 🛡️ プライバシー・セキュリティ

- すべての計算はお使いのブラウザ上（クライアントサイド）で完結します
- 入力データがインターネットに送信されることはありません
- データの保存はお使いの端末内（localStorage または JSONファイル）のみです

---

## 📋 仕様 / 前提条件

- 収入は「手取り額」（税金・社会保険料控除後）を入力してください
- 物価変動（インフレ/デフレ）は別途インフレ率で設定可能です
- 車両の日常維持費（ガソリン代・保険等）は「基本生活費」に含めてください
- 年金は22歳〜退職年齢まで継続加入を前提に簡易計算しています
